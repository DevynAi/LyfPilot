# Documentation

This directory contains supplementary documentation and assets for the LyfPilot showcase.

## Structure

```
docs/
├── assets/           # Images, banners, diagrams
│   └── (add screenshots and diagrams here)
└── README.md         # This file
```

## Adding Screenshots

Place screenshots in `docs/assets/` and reference them from the main README:

```markdown
![Dashboard](docs/assets/dashboard.png)
```

Recommended image sizes:
- Banner: 1600x400px
- Screenshots: 1200x800px
- Mobile screenshots: 400x800px
